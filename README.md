# 🏥 Diagnóstico Médico mediante Modelo Oculto de Markov (HMM)

Este repositorio contiene la implementación de una simulación estocástica para inferir el estado de salud de un paciente basándose en síntomas visibles.

---

## 📋 Descripción del Proyecto
El objetivo es aplicar los **Modelos Ocultos de Markov (HMM)** para modelar la evolución de un paciente a través de tres estados: **Saludable, Resfriado y Gripe**.

---

## 🛠️ Tecnologías y Librerías
Este proyecto utiliza:

- NumPy: cálculos probabilísticos y matrices de transición/emisión  
- Pandas: gestión de tablas de resultados  
- Matplotlib: generación de gráficas  
- ipywidgets: interfaz interactiva  

---

## 📊 Resultados clave

- Aplicación del algoritmo de Viterbi para estimar la secuencia de estados  
- Simulación del comportamiento del sistema en el tiempo  
- Análisis de probabilidades de transición  
- Visualización de evolución del estado del paciente
